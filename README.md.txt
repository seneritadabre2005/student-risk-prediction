# 🎓 Student Risk Prediction System

---

## 📌 Overview

The **Student Risk Prediction System** is a Machine Learning-based application designed to identify students who are at risk of poor academic performance.

The system analyzes student data such as attendance, internal marks, assignment scores, study hours, and sleep hours to predict whether a student is **At Risk** or **Not At Risk**.

It also includes a **real-time dashboard** for data visualization and better insights.

---

## 🎯 Objectives

- Predict student academic risk using Machine Learning  
- Analyze student performance data  
- Improve prediction accuracy using advanced algorithms  
- Visualize data using interactive charts  
- Simulate real-time data updates  
- Assist educators in identifying weak students early  

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, Streamlit  
- **Machine Learning Model:** Random Forest Classifier  
- **IDE:** VS Code  
- **Version Control:** Git & GitHub  

---

## 🤖 Machine Learning Approach

The project uses supervised learning for classification.

### Steps:

1. Data Generation and Collection  
2. Data Preprocessing  
3. Feature Selection  
4. Model Training (Random Forest)  
5. Model Evaluation  
6. Prediction and Deployment
